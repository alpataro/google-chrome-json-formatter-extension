# Privacy Policy  
**ALPATARO's JSON Formatter**

Last updated: 2025-12-25

---

## Overview

ALPATARO's JSON Formatter is a browser extension designed to improve the readability of raw JSON responses by formatting and syntax-highlighting JSON content directly within the user’s browser.

---

## Data Collection

ALPATARO's JSON Formatter does not collect any personal data.

Specifically:
- No personally identifiable information is collected
- No browsing history is collected
- No authentication or user identifiers are used
- No analytics or tracking mechanisms are implemented

---

## Data Processing

The extension processes page content locally within the user’s browser and only when valid JSON content is detected.

All formatting and syntax highlighting is performed locally. No data is sent to external servers.

---

## Network Requests

The extension does not make any network requests.

It does not communicate with external APIs, third-party services, or remote servers.

---

## Permissions

The extension does not request any Chrome permissions.

It operates solely as a content script to format JSON responses for improved readability.

---

## Third-Party Services

The extension does not use any third-party services, libraries, or analytics tools.

---

## Data Sharing

No data is shared with the developer or any third parties.

---

## Data Storage

The extension does not store any user data.

All processing is transient and occurs only during the active browser session.

---

## Changes to This Policy

If the extension’s functionality changes in a way that affects user privacy, this Privacy Policy will be updated accordingly.
